## 📂 Notable Projects:

### 🧠 [MediScan AI – Facial Skin Health Advisor](https://github.com/Mohamed-Arif-A/mediscan-ai)
<img src="https://raw.githubusercontent.com/Mohamed-Arif-A/mediscan-ai/main/demo/mediscan-preview.gif" width="100%" />

An intelligent AI-powered web app that detects facial skin issues like **acne, dry skin, blackheads, whiteheads**, and more.  
> **Stack:** Python · TensorFlow · MobileNetV2 · FastAPI · React · Gemini API  
🔍 Real-time facial analysis trained on a curated dataset. Fast API backend + smooth React UI.  
🎥 [Watch Demo](https://github.com/Mohamed-Arif-A/mediscan-ai/assets/demo.mp4)

---

### 🎯 [OpenCV Object Tracker](https://github.com/Mohamed-Arif-A/opencv-object-tracker)
<img src="https://raw.githubusercontent.com/Mohamed-Arif-A/opencv-object-tracker/main/demo/tracker-preview.gif" width="100%" />

Track selected objects in real-time from webcam using powerful OpenCV tracking algorithms like **KCF, CSRT, MOSSE, MIL**.  
> **Stack:** Python · OpenCV · NumPy  
📹 ROI selection with frame-by-frame object tracking. Ideal for **surveillance, automation, and gesture control**.  
🎥 [Watch in Action](https://github.com/Mohamed-Arif-A/opencv-object-tracker/assets/demo.mp4)
